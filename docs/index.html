Get it

Install in 3 steps

No computer needed on your end — just the file and your own key.

↓ Download SlyOS.apk

Android 10+ · ~50 MB

On your Android phone

  1. Tap Download, then open the file. Allow "install from unknown apps" once, then Install.
  2. Open SlyOS → the setup wizard takes your Anthropic key, then optional profile & imports.
  3. Press Home → SlyOS → Always. It's now your home screen.
🔑 You bring your own Anthropic API key.

Get one free at console.anthropic.com → API Keys → Create Key (starts with sk-ant-). Usage runs on your own account, and SlyOS stores the key only on your phone — never shared.
