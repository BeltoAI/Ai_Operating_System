Setup & Help

Get SlyOS running in 10 minutes

Follow these in order. Each step is written plainly, with the exact taps for Samsung (One UI) and most Android phones. Nothing here leaves your phone.

Part 1 · Install

Download the app

On your phone, tap Download SlyOS.apk. When it finishes, tap the file to open it.

Allow install & tap Install

Android will ask to allow installing from your browser/files app — tap Allow (this is one-time), then tap Install, then Open.

Make SlyOS your Home screen

This is what turns your phone into SlyOS. In the app tap “Set SlyOS as my Home app,” or set it manually:

Settings → search “Default apps” → Home app → SlyOS

Part 2 · Give it a brain

Add an API key (free option)

SlyOS needs a model key to think. Gemini is free: make one at aistudio.google.com/app/apikey, then in SlyOS go to Memory → gear (top-right) → “API keys & model”, paste it, and tap Save & check. The dot turns ✓ Valid when it's accepted. You can also add Claude or OpenAI keys the same way.

Part 3 · Turn on the powers

Permissions, one by one

Each unlocks a feature. After any reinstall, Android resets these — you'll need to re-grant them.

🔔 Notification access Now feed + auto-reply

Lets SlyOS see and reply to your messages. This is the #1 thing people forget.

Settings → search “Notification access” → SlyOS → on → Allow

Manual: Settings → Apps → ⋮ → Special access → Notification access → SlyOS.

♿ Accessibility Action layer

Lets SlyOS tap/scroll on your behalf and read the current screen. It never taps anything final (Send/Pay/Post) without you.

Settings → Accessibility → Installed apps → SlyOS → on

🪟 Display over other apps Nav panel

Lets the SlyOS nav bar float over other apps (rolling out).

Settings → search “Appear on top” / “Display over other apps” → SlyOS → on

🔗 Connect Google Docs · Gmail · Drive backup

Enables real Google Docs/Sheets/Slides, Gmail send, calendar, and the all-important brain backup to Drive.

SlyOS → Memory → Connections → Connect

Part 4 · Protect your brain

Turn on brain backup

Your on-device memory is precious — back it up so a reinstall or new phone never loses it. After connecting Google, open Memory → gear → scroll to “🛡 Brain backup” (bottom) → Back up now. You want to see “Drive ✓”. Automatic backup is on by default and also drops a copy in your Downloads folder.

Troubleshooting

Common issues

My Now feed is empty / notifications stopped showing

This almost always means Notification access got turned off — Android revokes it whenever the app is reinstalled or updated a certain way. Re-enable it: Settings → search “Notification access” → turn SlyOS back on. Then text yourself; it should appear within a second.

I updated the app but nothing looks different

The new version installed, but the app was still running the old screen from memory. Fully close SlyOS (recents → swipe it away, or Settings → Apps → SlyOS → Force stop) and reopen it. The Memory settings screen shows a small “build” badge at the top so you can confirm you're on the latest.

Is my API key actually working?

Open Memory → gear → “API keys & model” and look at the colored dot next to each key: grey “Not set,” orange “Checking…,” green “✓ Valid,” or red “✗ Invalid.” SlyOS pings the provider to confirm, so you're never guessing. If it's red, re-copy the key (no extra spaces) and Save & check again.

Where do my API keys go? Are they safe?

Keys are stored only on your phone and are never uploaded to us. All your keys live in one place: Memory → gear → “API keys & model.”

I got a new phone / reinstalled — how do I get my brain back?

Reconnect Google, then Memory → gear → “🛡 Brain backup” (bottom) → Restore from Drive. It downloads your latest backup and relaunches. (This only works if backup was on before — it's on by default.)

Auto-replies aren't sending (or are only drafting)

By design, unknown/first-time senders are drafted, not sent, and each app has its own level (off / draft / full) under Memory → gear → “Per-app responses.” Set an app to “full” to allow auto-send. Risky drafts (links, money, secrets) are always held for your review.

How do I go back to my normal launcher?

It's fully reversible: Settings → Default apps → Home app → pick your old launcher.
